# I do Too Crew - Content Template System

## Purpose

This document defines repeatable content templates for social posts, reels, and watermark usage. It is intended to keep output consistent even before a full brand style guide is finalized.

## Core Principle

The system should be modular.

That means:
- same structure
- flexible colours and assets later
- repeatable layouts
- consistent CTA placement
- optional watermark usage

## Template Categories

### 1. Static Post Templates
Use for:
- promotional offers
- quotes
- package highlights
- testimonials
- event inspiration

#### Static Template A: Hero Offer Post
Use for:
- package promotion
- lead generation

Structure:
- strong headline
- supporting line
- service context
- CTA at bottom
- small logo watermark in corner

Example use:
- Elegant wedding planning for unforgettable days.
- Get a quote today.

#### Static Template B: Quote / Emotion Card
Use for:
- emotional branding
- wedding storytelling
- aspirational captions

Structure:
- image-led background
- short emotional line
- subtle logo mark

#### Static Template C: FAQ / Tip Card
Use for:
- education posts
- parent reassurance
- planning tips

Structure:
- question headline
- 3 to 5 bullet style answers
- CTA footer

#### Static Template D: Testimonial Card
Use for:
- social proof
- trust-building

Structure:
- short client quote
- optional event label
- subtle brand mark

## 4. Watermark System

## General rule
The watermark should support recognition, not distract from the content.

### Watermark placement options
- bottom right corner
- top right corner for simple layouts
- very subtle center overlay for selected stills only

### Watermark usage by content type

#### Static posts
- small corner watermark
- low opacity if over image
- solid version if placed on clean background block

#### Carousels
- only on first and last slide, or all slides if very subtle

#### Reels and short videos
- subtle small corner watermark throughout
- stronger logo lockup at end card

### Watermark rules
- never cover faces
- never cover key decor details
- never overpower the CTA
- keep placement consistent across a content family

## 7. Content Production Rules
- Every asset should belong to a content pillar
- Every asset should map to a platform
- Every asset should have a CTA intention
- Watermark use should be optional but documented in Airtable
- Reel templates should prioritize the first 2 seconds for hook strength
